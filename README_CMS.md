# Netlify CMS - Guía de Uso

## ¿Cómo acceder al panel de administración?

1. Ve a: `https://tusitio.netlify.app/admin`
2. Inicia sesión con tu cuenta de Netlify o GitHub
3. ¡Ya puedes editar el contenido!

## ¿Qué puedes editar?

### 1. Próximas Fechas (Tour)
- Agregar nuevas fechas de conciertos
- Editar fechas existentes
- Cambiar ciudad, venue, estado, etc.

### 2. Eventos Timeline
- Agregar nuevos eventos a la trayectoria
- Editar eventos existentes
- Subir fotos para cada evento

### 3. Contenido General
- Tagline principal
- Descripción de "About Us"
- Información de contacto
- Links de redes sociales

## Notas Importantes

- Los cambios se guardan automáticamente
- Puede tomar unos minutos en aparecer en el sitio
- Necesitas tener acceso al repositorio Git conectado a Netlify
